TCGA Variant calling

a smidge of background

Cancer is a disease of the genome. Given the genome sequences of an indivduals normal tissue and their tumor, we want to tell what went wrong: What particular mutations are present in patients tumor but not their normal sequence? Our interest is not mearly intellectual. A tumors mutational landscape may have implications on patient prognosis and on the best course of treatment.

There are many somatic variant calling tools available to answer the above question. These tools generally take as input tumor-normal matched sequence pair, the mutations unique to the tumor (Callers generally focus on single nuceotide polymorphisms or SNPs, and insertions and deletions or INDELS). You can read more about available tools and pipelines here, and here

The largest repository of tumor genomics information is The Cancer Genome Atlas (TCGA), which contains OMICS data for 33,000 cases. Each TCGA sample is subject to four variant calling pipelines, which are representative of the most reliable callers currently available. Each pipeline is based on a different caller (Mutect, Varscan2, SomaticSniper, and you can read about all the gory pipeline details here.

Suprisingly, the results of these four pipelines show relatively little consistency. I downloaded somatic MAF files for each pipeline (which have SNP and indel calls), and made this venn diagram venn) of all the calls for the 1100 or so BRCA samples on TCGA:

read.table.with.name <- function(df.name, levels, ...){
  #read in df table, with name of file as factor
  df <- read.table(df.name, ...)
  df$caller <- factor(df.name, levels=levels)
  df
}

IOU <- function(...){ 
  #find size of intersection over union for arbitrary # of vectors
  args <- list(...)
  intersect_ <- reduce(args, intersect)
  union_ <- reduce(args, union)
  return(length(intersect_)/length(union_))
}

IOU_df <- breast_df %>% 
  group_by(Tumor_Sample_Barcode) %>% 
  do(caller_vector=.$caller, start_position_vector=.$Start_Position) %>% ungroup %>% 
  dplyr::mutate(start_list = map2(.x = start_position_vector, .y = caller_vector, .f = ~ split(x = .x, f = .y))) %>% mutate(IOU= map_dbl(.x = .$start_list, .f = ~do.call(IOU, .x))) %>%
